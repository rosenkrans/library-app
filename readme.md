## Coding languages and libraries

* HTML
* CSS
* JavaScript
* Python
* Django
* AJAX
* React
* React Bootstrap

## Features

* Add, view, edit, and delete a category name
* View list of categories
* Add, view, edit, and delete a book
* View list of books
* Add, view, edit, and delete a member
* View a list of members
